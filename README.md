## Instruções de como rodar o projeto.

### Instalação

```bash
$ pnpm install
```

### Executando o app

```bash
# development
$ pnpm run start

#or
$ docker-compose build
$ docker-compose up

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### Teste

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
